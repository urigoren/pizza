# Pizza Triangles

This is a simple web application that visualizes pizza-like triangular slices on a canvas with various customizable parameters, including animation and wavy lines.

## Screenshot
![Screenshot of Pizza Triangles Application](screenshot.png)

## Features
- **Number of Triangles:** Adjust the count of pizza slices.
- **Rotation:** Rotate the entire pizza figure.
- **Fill Color:** Choose the color for the pizza slices.
- **Line Color:** Select the color of the wavy lines separating the slices.
- **Canvas Color:** Set the background color of the drawing canvas.
- **Amplitude:** Control the intensity of the wave effect on the lines.
- **Frequency:** Adjust how many waves appear along each line.
- **Rate:** Define the speed of the animation (seconds for a full phase shift cycle).


## How to Use
1.  **Start the web server:** Ensure a local web server is running in the project directory (e.g., `python3 -m http.server 8000`).
2.  **Open in browser:** Navigate to `http://localhost:8000` in your web browser.
3.  **Adjust controls:** Use the input fields on the left to customize the pizza's appearance and animation.
4.  **Animate:** Click the "Animate" button to start the continuous animation with the current settings. You can change settings and click "Animate" again to apply the new parameters.
